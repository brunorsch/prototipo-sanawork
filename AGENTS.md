# SanaWork - Protótipo
Este projeto é um protótipo focado em demonstrar a viabilidade e valor de mercado da startup SanaWork (ou nome de marca parametrizado).
- **Simplicidade e Velocidade:** Código limpo, sem complexidade desnecessária.
- **Sem CRUD:** Sem operações de banco de dados ou persistência. Usar estados locais (React state/context) para simular.
- **Funcionalidades Diferenciadas:** Focar em demonstrar as propostas de valor únicas da startup:
  1. Painel de sensibilização para empresas com métricas de produtividade/INSS.
  2. Triagem e histórico de saúde dos funcionários com anonimização.
  3. Relato por voz do colaborador analisado por IA.
  4. Indicação inteligente de folga-saúde e profissionais indicados.
- **Massa de Dados Fictícia:** Utilizar dados de mock realistas para listagens de funcionários, históricos de triagens e estatísticas.
- **Parametrização:** Permitir customizar o nome do projeto (marca) e as credenciais de API da IA (compatível com a API de Chat da OpenAI).
